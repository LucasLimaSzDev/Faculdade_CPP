# Contribution Guide

This repository is an academic activity archive maintained by Lucas Lima.

## Organization

Use one folder per semester, discipline or assignment list:

```text
assignments/
  semester-01/
    list-01/
      README.md
      solution-file
```

## Activity README

Each activity folder should explain:

- objective
- input
- output
- approach
- status

## Privacy

Do not commit private documents, student IDs, full class lists, credentials,
screenshots with personal data, or files from the university portal that cannot
be redistributed.
