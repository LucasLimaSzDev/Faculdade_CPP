## Overview

Repository for college C++ programming activities, exercises and study notes.

This repository is organized as a clean academic archive and portfolio. Each
activity should be easy to find, read, run and review later.

## Focus

- C++ fundamentals
- Input and output
- Conditionals and loops
- Functions and arrays
- College assignments and practice exercises

## Structure

~~~text
assignments/
  semester-01/
    list-01/
      README.md
      solution-file
docs/
  INDEX.md
templates/
  activity/
~~~

## How To Add A New Activity

1. Create a folder inside the assignments/semester-01/ directory.
2. Use a short folder name, such as list-02, exam-practice or arrays.
3. Add a README.md explaining the activity.
4. Add the solution source file.
5. Keep filenames lowercase and descriptive.

## Running

Compile each activity from its own folder. Example:

~~~powershell
g++ src/main.cpp -o build/activity.exe
.\build\activity.exe
~~~
